# Specification

## Summary
**Goal:** Add detailed interactive feature showcase cards to the Gambia Market blueprint app, covering all major app features as static/presentational UI mockups.

**Planned changes:**
- Add Phone Number Registration with OTP showcase: step-by-step flow with phone input, OTP input, name/location fields, and validation states
- Add Listing Creation showcase: multi-step walkthrough with photo upload, category selector, price input, condition selector, and location tagging
- Add Home Feed showcase: mock listing grid with 3–4 sample cards showing image, title, price, location badge, and distance indicator
- Add Search & Filters showcase: search bar mockup with category dropdown, price range inputs, and condition checkboxes
- Add In-App Chat showcase: conversation thread mockup with message bubbles, listing context header, timestamps, and message input bar
- Add Verified Seller Badge showcase: verification steps card (ID upload → review → badge awarded) with sample seller profile card displaying badge, rating, and transaction count
- Add Delivery & Meet-Up showcase: side-by-side comparison card for delivery request (address, cost, partner) and meet-up (location, time) options
- Add Mobile Money Payments showcase: payment selection mockup with Afrimoney, QMoney, and Cash on Delivery options, plus transaction status badges
- Add Trust & Safety System showcase: 2×2 grid of mini-cards for star ratings, written reviews, report system, and block user dialog
- Add Boosted & Promoted Listings showcase in Monetization section: standard vs. boosted listing comparison with pricing tiers in GMD
- Store all new showcase content as structured TypeScript data in `frontend/src/data/` and render dynamically
- Add app screen mockup image and payment methods illustration image

**User-visible outcome:** Visitors to the Gambia Market blueprint app can explore richly detailed, visually consistent showcase cards for every major feature of the planned app, including registration, listings, chat, payments, and trust systems.
